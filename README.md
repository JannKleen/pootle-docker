Pootle Docker Container
=======================

This is the source for **jannkleen/pootle** in the docker index.

Installation
------------

- The easy way:

    ```
    docker pull jannkleen/pootle
    ```

- The other way:

    ```
    git clone https://github.com/JannKleen/pootle-docker.git
    cd pootle-docker
    docker build .
    ```
